# Glacier*

Native macOS file explorer and editor. Built for macOS 26 (Tahoe).

> \* **No longer actively supported.** The developer is stepping away from this project to focus on other goals for now. The code remains here for reference and may be revisited in the future.

## Excalidraw

Excalidraw loads directly from `excalidraw.com` in an embedded WebView. Requires internet. Always up to date automatically — no maintenance needed.

Drawings are auto-saved back to the `.excalidraw` file on disk every 1.5 seconds after changes.
